Creative Coding with p5.js

p5.js is an open-source JavaScript library that makes creative coding accessible. Inspired by Processing, it gives you a full set of drawing primitives, animation loops, and input handling — so you can focus on making things rather than wiring up a renderer.

  • Canvas 2D & WebGL Draw shapes, images, and 3D geometry
  • 60 fps loop Animate with setup() and draw()
  • Interaction Mouse, keyboard, touch — all built in
  • Math & Noise Vectors, Perlin noise, trigonometry helpers
